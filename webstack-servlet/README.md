# WEB Stack

## 1. Servlet
> Servlet是Java Web开发中处理HTTP请求和响应的组件。



### 1.1 HttpServletRequest
> HttpServletRequest对象封装了HTTP请求的所有信息。
> 获取请求参数：
> 1. getParameter()
> 2. getParameterMap()
> 3. getHeaders()
> 4. getCookies()

### 1.2 HttpServletResponse
> HttpServletResponse对象封装了HTTP响应的所有信息。
> 设置响应数据：
> 1. setHeader()
> 2. setContentType()
> 3. setCookie()
> 4. getOutputStream()
